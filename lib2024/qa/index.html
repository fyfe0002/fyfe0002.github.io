QA for LIB2024

  1. Qus: What are the three components of Web?

    Ans: The three components of the web are HTML, CSS, and JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a software that allows an ongoing record of files over a period of time, so that threre is a history of the versions. GitHub, which we use, is an example of version control software.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The tag title is used to create the title of a web page. <title>Name of page</title> is an example.

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tag p is used to create a paragraph in html. <p>Paragraph</p> is an example
